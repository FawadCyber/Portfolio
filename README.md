# Fawad Naseem — Portfolio

A responsive portfolio for an AI Automation Engineer, featuring production n8n systems, custom MCP infrastructure, API integrations, a 20-second Remotion showreel, and a clickable proof library.

The site is fully static. GitHub Pages can serve it directly with no build step.

## Verified workflow cases

Six proof tiles are visible directly in the main proof section. Each tile opens a detailed modal with its architecture, system flow, evidence notes, earlier proof videos where available, and a responsive screenshot gallery.

- Pinterest MCP Server
- MP Executive Dashboard
- FACON FBA Weekly Dashboard
- Sell Bills to Odoo Sales Orders
- Amazon Buyer Message Fetching
- YouTube MCP Server

The public package contains 29 proof files. Buyer-identifying screenshots, raw n8n exports, webhook IDs, and credential references are intentionally excluded from the deployable site.
